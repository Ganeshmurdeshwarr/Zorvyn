# 💰 FinTech Dashboard

A modern FinTech dashboard built with React that provides insights into user spending, transactions, and financial analytics.

---

## 🚀 Live Demo

👉 https://zorvyn-9ygpz4a6p-ganesh-devadigas-projects.vercel.app/

---

## 📌 Features

* 📊 Interactive dashboard with charts (spending, categories, trends)
* 👥 User selection & dynamic data rendering
* 🔍 Search functionality
* 📅 Filter & sort transactions (date, category, etc.)
* 📈 Visual analytics (Pie chart, comparisons)
* 📱 Fully responsive design
* ⚡ Fast and optimized UI

---

## 🛠️ Tech Stack

* **Frontend:** React.js, Tailwind CSS
* **Charts:** Recharts
* **State Management:** Context API
* **Icons:** React Icons
* **Build Tool:** Vite
